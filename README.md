# Hand Sign - 轻量化数字签名与证件扫描工具

Hand Sign 是一款基于纯前端技术 (HTML, CSS, JavaScript) 开发的轻量化数字工具集。它提供干净、极简的交互界面，支持在线手写签名采集、图片背景透明化以及专业的证件扫描功能。

## ✨ 核心特性

### 1. 流畅的手写签名
- **平滑笔迹**：使用 Canvas 和贝塞尔曲线实现细腻流畅的手写体验。
- **个性化定制**：支持自定义笔触颜色、实时调整线宽。
- **便捷操作**：支持多级撤销 (Undo) 和一键清除画板。
- **透明导出**：一键导出免抠图的透明背景 PNG/WebP，或矢量 SVG 格式。

### 2. 智能证件扫描 (New!)
- **透视矫正**：支持四点自由拖拽，自动将倾斜拍摄的证件矫正为正视图。
- **背景漂白**：内置智能算法，自动去除文档背景阴影和不均匀光照，实现扫描仪级别的纯净效果。
- **多种模式**：提供彩色增强、灰度扫描、黑白文档三种处理模式。
- **实时调优**：支持亮度、对比度、锐度的实时调整与预览。
- **多格式导出**：支持导出为高清图片或标准 A4 尺寸的 PDF 文件。

### 3. 图片去背景
- **一键抠图**：基于颜色容差算法，快速去除签名图片或简单背景图片的底色。
- **多端兼容**：完美适配 PC 鼠标和移动端触摸操作。

## 🚀 快速启动

本项目为纯前端项目，无任何复杂依赖，您可以直接在本地运行。

### 方法一：使用 Node.js 快速启动（推荐）
在项目根目录运行以下命令开启一个本地页面服务器：
```bash
npx serve . -p 3456
```
然后在浏览器中访问 `http://localhost:3456`。

### 方法二：使用 VS Code Live Server
1. 在 VS Code 中打开本项目。
2. 安装 `Live Server` 扩展。
3. 打开 `index.html` 文件，右键选择 **Open with Live Server**。

### 方法三：直接打开
直接在浏览器中双击打开 `index.html` 即可使用大部分功能。

## 🛠 技术栈

- **HTML5** (Canvas 2D API)
- **Vanilla CSS3** (CSS Variables, Flex/Grid 布局)
- **Vanilla JavaScript** (无框架依赖，高性能像素级图像处理)
- **第三方库**: [jsPDF](https://github.com/parallax/jsPDF) (用于 PDF 导出)
- **字体**: [Inter](https://fonts.google.com/specimen/Inter)
