# Hand Sign - 轻量化数字签名采集工具

Hand Sign 是一款基于纯前端技术 (HTML, CSS, JavaScript) 开发的轻量化数字签名采集工具。提供干净、极简的画板，支持在网页端通过鼠标或触摸屏快速创建个人电子签名，并一键导出透明底的高清 PNG 签名图片。

## ✨ 核心特性

- **流畅的签名体验**：使用 Canvas 和贝塞尔曲线实现平滑的手写笔迹。
- **多端兼容**：同时支持鼠标点击拖拽和移动设备的触摸操作。
- **个性化笔触**：
  - 自定义笔触颜色
  - 实时预览调整线宽粗细
- **高效还原与操作**：支持对签名过程的撤销和整个画板的一键清除。
- **透明底导出**：一键将签名导出为免抠图的透明背景 PNG 图片，完美适用插入电子合同或 PDF 测试。
- **视觉体验优秀**：基于 Flat Design 风格，UI 界面干净整洁，色彩采用专业的青色主体，响应式设计适配各类尺寸屏幕。

## 🚀 快速启动

本项目为纯前端项目，无任何复杂依赖，您可以直接在本地运行。

### 方法一：使用 VS Code Live Server（推荐）
1. 在 VS Code 中打开本项目。
2. 安装 `Live Server` 扩展。
3. 打开 `index.html` 文件，右键选择 **Open with Live Server**，或者点击右下角的 **Go Live**。

### 方法二：使用 Node.js 快速启动
在项目根目录运行以下命令开启一个本地页面服务器：
```bash
npx serve . -p 3456
```
然后在浏览器中访问 `http://localhost:3456`。

### 方法三：直接双击
如果您不需要某些严格的服务器环境特性，也可以直接在浏览器中双击打开 `index.html`。

## 🛠 技术栈

- **HTML5** (Canvas 绘图 API)
- **Vanilla CSS3** (使用 CSS 变量，响应式布局)
- **Vanilla JavaScript** (无框架依赖，轻便高效)
- **字体**: [Inter](https://fonts.google.com/specimen/Inter)
